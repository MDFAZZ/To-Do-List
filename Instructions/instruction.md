Assignment : To-Do List Project with APIs and Templates
 

Objective:

Develop a web application using Python and Django/Flask/FastAPI for managing a To-Do list. The application should provide RESTful APIs for CRUD operations on tasks, utilize templates for rendering user interfaces, and store task data in a database.

 

Requirements:

Set Up Django App with API Endpoints:

Create a Django/Flask application with routes for API endpoints to handle CRUD operations on tasks.

Create a task

Retrieve tasks

 

Database Setup:

Use SQLite or any other database of your choice.

Create a table to store tasks. The table should include fields such as task title, description, due date, and status.

  

Build Task API:

Implement API endpoints for performing CRUD operations on tasks.

The API should accept and return JSON data for task details.

 

Use Templates for Web Interface:

Create HTML templates 

Design templates for:

Displaying the list of tasks

Adding a new task


Integrate Templates with API Endpoints:

Use the templates to render forms and display the list of tasks in the web application.

When a user interacts with the web interface, make use of the corresponding API endpoint for CRUD operations.

 

Testing:

Implement basic tests for each API endpoint to ensure proper functionality.

Use a testing framework like pytest to automate the testing process.

 

API Documentation:

Create documentation for the APIs, including endpoint descriptions, expected request/response formats, and any authentication mechanisms.
